Lecture Viewer API

LectureViewer API will be utilized by the React front end and will allow for CRUD operations by being the middle man for the database.

NOTE: This is an incomplete API at the moment. The base url calls are all defined but not all of the request/response data is finalized

API Responses

These are examples of the responses that one can expect to recieve from the API

A Successful Response

This shows a response example of what a correctly used api with no errors would look like

    
{
    status : "success",
    data : {
        "posts" : [
            { "id" : 1, "title" : "A blog post", "body" : "Some useful content" },
            { "id" : 2, "title" : "Another blog post", "body" : "More content" },
        ]
     }
}

A Failed Response

This shows a response example of what an improperly formatted api call looks like

    
{
    status : "fail",
    data : { "title" : "A title is required" }
}

An Error Response

This shows a response example of what an internal error would return

    
{
    status : "error",
    message : "There was a database connection error"
}
